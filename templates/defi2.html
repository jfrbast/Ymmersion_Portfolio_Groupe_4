CHALLENGE N°2 : Guess the number !

Difficulty :

Challenge rule:

Create a program that generates a random number. The user must guess the number.
At each attempt, the program indicates whether the number entered is too small or too large, until the user finds the right answer.

Stages of the challenge and reflection :

To complete this challenge, I decided to take it one step at a time.
First, I asked myself where to start.

  • Step 1: The program asks us to choose a number between 0 and 100.
    Create a solution variable to generate a random number between 0 and 100.
  • Step 2: We create a Try variable that is a number and then we make it show in the terminal.
  • Step 3: If Try is smaller than the solution, the program will return "It’s bigger!"
    Conversely, if Try is larger than the solution, the program will return "It’s smaller".
  • Step 4: If we manage to find the right number, the program will say "Well done!"
